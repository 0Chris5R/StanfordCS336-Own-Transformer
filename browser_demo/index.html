Projects

Browser LM Language Model

A personal language model trained from scratch that runs entirely in the browser. 70.7M parameters with custom BPE tokenizer, Flash Attention Kernel and Muon Optimizer. Trained with DDP on 2x T4 GPUs for 450M tokens. SFT on 1.1k synthetic Q&A and refusal pairs with short classification CoT. deployed via ONNX Runtime WebAssembly. Demonstrates end-to-end ML engineering from synthetic data generation to deployment.

PyTorch, ONNX, WebAssembly, BPE

Complete decoder-only transformer using only NumPy and PyTorch primitives. Implements RMSNorm, RoPE, SwiGLU, Multi-Head Attention. Includes custom BPE tokenizer with parallel processing, and AdamW/Muon optimizers from scratch.
